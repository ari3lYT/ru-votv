# Polish pass 001

Первый ручной проход по самым заметным машинным ошибкам.

Что исправлено:

* Описания предметов и подсказки с `LMB` / `RMB` / `MMB`: заменены на `ЛКМ` / `ПКМ` / `СКМ`.
* Вручную переписаны самые повторяющиеся строки: канистра, гвозди, пиво, цифровая камера,
  разбрызгиватель, дробилка, пульт TV, мойка, фонарик, цемент, термометр, бирка, сигареты,
  маска/баллоны для дайвинга, нож, гирлянды и несколько prop-строк.
* `Performance cost` больше не переводится как `Стоимость исполнения`.
* Восстановлены служебные команды help-разметки: `/imagesize` и `/align`.
* Убраны zero-width и non-breaking пробелы из русских строк.

Проверки после прохода:

* `RMB/LMB/MMB` в русских строках: `0`.
* Сломанные `/imagesize` / `/align`: `0`.
* Несовпадения `{...}` и markup-токенов: `0`.
