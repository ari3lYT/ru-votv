# Глоссарий

Основные решения для единообразия:

* `drive` как носитель сигнала -> `диск`.
* `drive box` -> `коробка для дисков`.
* `drive rack` -> `стойка для дисков`.
* `drive slot` -> `слот для диска`.
* `floppy disc` -> `дискета`.
* `ZIP drive` / `ZIP disk` -> `ZIP-диск`.
* `signal` -> `сигнал`.
* `data tape` -> `лента данных`.
* `tape case` -> `футляр для лент`.
* `server` -> `сервер`.
* `server upgrade` -> `улучшение сервера`.
* `upgrade` -> `улучшение`, не `обновление`.
* `ATV` -> `квадроцикл`.
* `drone` -> `дрон`.
* `scrap` -> по контексту: `лом`, `обломок`, `кусок`, `материал`.
* `prop` -> `объект`, не `реквизит`.
* `satellite report` -> `отчёт по спутнику`.
* `satellite server` -> `спутниковый сервер`.
* `store` как вкладка/место покупки -> `магазин`.
* `points` как награда/валюта задач -> `очки`.
* `Help & Info` / `Hints and Tips` как единое меню -> `Справка и советы`.
* `Funny` как настройка -> `Веселье`.
* `Custom Content` -> `[Пользовательский контент]`.
* `RHI`, `TAA`, `AA`, `SSR`, `SSGI`, `HDR`, `VSync`, `FOV` -> оставляем латиницей.
* `Screen Space` -> `экранное пространство`.
* `Bloom` -> `bloom`.
* `motion blur` -> `размытие в движении`.
* `render distance` -> `дальность прорисовки`.
* `upscaling` / `upsampling` -> `апскейлинг`, если это название настройки.
* `performance cost` -> `нагрузка на производительность`.

Что не переводим без причины:

* имена файлов, пути, URL;
* команды консоли;
* технические id вроде `server_repair_cluster`;
* названия брендов, имена персонажей и мемные строки без контекста;
* `FPS`, `HDR`, `TAA`, `SSR`, `SSGI`, `USB`, `WASD`.
