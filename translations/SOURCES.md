# Источники перевода

## Game.locres

База основного перевода игры импортирована из:

- Manual Russian Translation by Antoha256M и MrLololoshenka
- Thunderstore: https://thunderstore.io/c/voices-of-the-void/p/Antoha256M/Manual_Russian_Translation/
- Версия пакета: `1.1.1`
- Дата пакета: `2026-04-27`
- Профиль автора из манифеста: https://discord.com/users/887093938685616138

Локально поверх базы исправлены очевидные опечатки:

- `сломманого` -> `сломанного`
- `радивышки` -> `радиовышки`
- `ОтменА` -> `Отмена`
- `Нищюган` -> `Нищуган`
- `25 баллов` -> `25 очков`
- восстановлены rich-text команды `/imagesize`, `/align`, `/padding`, которые нельзя переводить как обычный текст
