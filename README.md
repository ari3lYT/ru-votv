# Voices of the Void — русский перевод

Репозиторий с полной русификацией Voices of the Void (готовые патчи `.pak` для уже установленной игры).

## Что внутри
- `ZZ_GameRuPatch_P.pak` — перевод игрового контента (Game.locres).
- `ZZ_EngineRuPatch_P.pak` — перевод интерфейса движка (Engine.locres). Если не нужен перевод движка, этот файл можно не класть.

## Как установить
1. Откройте папку игры и перейдите в `VotV/Content/Paks/` (пример: `/home/ariel/games/a09_0015/WindowsNoEditor/VotV/Content/Paks/`).
2. Скопируйте туда нужные `.pak` из этого архива. Важно не переименовывать файлы — от этого зависит порядок подключения патча.
3. Запустите игру — патчи подтянутся автоматически. Никаких замен исходных файлов не требуется. Удалите `.pak`, чтобы откатить перевод.

## Как делался перевод
- Базовые тексты извлечены из оригинальных `.locres`.
- Перевод выполнен автоматически через ChatGPT‑5.1, без ручных правок — возможны неточности и «сырость».
- Собраны в `.locres` и упакованы в `.pak` с корректным mount point, чтобы перекрывать оригинал при запуске.
